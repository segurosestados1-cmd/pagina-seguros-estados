# Guía Rápida: Cómo Guardar el Logo

## Pasos Simples:

1. **Obtén el logo:**
   - Desde el sitio: https://www.segurosdelestado.com/
   - O desde donde lo tengas guardado

2. **Copia el archivo:**
   - Copia el archivo del logo (logo.png, logo.svg, o como se llame)

3. **Pégalo en la carpeta:**
   - Ve a la carpeta del proyecto
   - Abre la carpeta `public`
   - Pega el logo ahí
   - Si no se llama `logo.png` o `logo.svg`, renómbralo

4. **Ruta completa:**
   ```
   C:\Users\usuario\Documents\aplicacion\seguros del estado\seguros del estado\public\logo.png
   ```

## Formatos Aceptados:
- ✅ `.png` (recomendado)
- ✅ `.svg` (mejor calidad)
- ✅ `.jpg` (si no tienes otros formatos)

## Verificación:
Una vez guardado, el logo debería aparecer automáticamente en el header del sitio.
